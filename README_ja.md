# X Likes Viewer

Twitter / X の「いいね」を JSON ファイルから読み取り、効率的に閲覧するための試験的なアプローチ。

## 機能

- 検索（マイナス検索・&検索対応）
- ソート（新しい順・古い順・いいね数順）
- 日付指定
- 日・英・中・韓の四ヶ国語対応
- 複数の JSON ファイルのキャッシュ保存（履歴から選択可）
- テーマ選択：ライト / ダーク / AMOLED
- PC・スマホ両対応
- Twitter API 不使用。ブラウザ上で完結し、入力情報は外部に送信されない
- HTML ファイルを保存すればローカル環境でも完全動作
